### 📝 Notes

- List all database in PostgreSQL: `\l`
- Move inside a database: `\c`
- Show table in database: `\dt`
- Create a database called *todo_db*: `CREATE DATABASE todo_db`
- Create a table called *todo*: `CREATE TABLE todo(...)`
